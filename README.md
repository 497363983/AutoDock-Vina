# AutoDock Vina
Molecular docking and virtual screening program

## Documentation

Build documentation with Sphinx (https://www.sphinx-doc.org/en/master/usage/installation.html)

```bash
$ cd docs
$ make html
# Open index.html located in docs/build/html
```
